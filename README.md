# TAK SMS Forwarder - Kotlin Final Fixed V2

اصلاحات این نسخه:
- منوی پایین با fixed + safe-area + z-index اصلاح شد.
- Toggle وضعیت سرویس بلافاصله ON/OFF می‌شود و متن فعال/غیرفعال هماهنگ است.
- عنوان «نوع فوروارد» حذف شد؛ تصمیم ارسال از روی مقصدهاست.
- مرحله مقصدها: یک کادر چندخطی شماره مقصد و یک کادر چندخطی ایمیل مقصد.
- هر خط یک مقصد است؛ کاما پیشنهاد یا استفاده نمی‌شود.
- خلاصه ویزارد شماره‌ها و ایمیل‌های مقصد را زیر هم نشان می‌دهد.
- «وضعیت فیلتر» از ویزارد حذف شد؛ فیلتر جدید پیش‌فرض فعال است.
- دکمه «افزودن SMS تست» از تنظیمات حذف شد.
- پیام فورواردشده دقیقاً متن اصلی SMS است؛ هیچ Rule/Sender/SMS Forwarded اضافه نمی‌شود.
- پیام بدون Match با فیلتر فعال، اصلاً لاگ نمی‌شود.
- در جزئیات پیام، اگر موفق باشد عنوان «نتیجه» می‌آید و فقط در خطای واقعی عنوان «خطا» می‌آید.
- SMS و Email می‌توانند همزمان ارسال شوند؛ اگر هر دو مقصد پر باشند هر دو انجام می‌شود.
- فایل‌ها Kotlin هستند؛ Java ندارد.
